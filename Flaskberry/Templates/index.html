Flaskberry

Simple web app to control Raspberry Pi via Controlberry


For more info visit Flaskberry repository on github


    After setup you are able to:

  • To see values from Adafruit's sensors AM2302, DHT22, DHT11
  • Take photo from PiCamera
  • Take photos in intervals and store them in database
  • Measure distance from HC-SR04
  • Control LEDs or anything else with pulse width modulation
  • Turn on or turn off selected GPIO pins
  • Check temperatures from DS18B20 sensors
